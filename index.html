About Math Hub

Math Hub is a student-led initiative designed to make learning math more engaging, collaborative, and accessible. We believe that peer-to-peer support is one of the most effective ways to learn, which is why we've created this platform to connect students with resources and tutors.

Our Mission

Our mission is to build a supportive online community where students can find the help they need, track their progress, and feel motivated to succeed in mathematics. We aim to provide high-quality, easy-to-understand resources that supplement classroom learning.

Planning & Development (FBLA)

This website was designed and developed for the FBLA Website Design competition. Our process focused on user-centered design principles to meet the 2025-2026 topic: "Design to Learn: Build a Student Learning Hub."

  • Planning: We identified the core needs of a student user: access to resources, progress tracking, and personalized help. This led to the three key features: the Resources section, the Student Dashboard (post-login), and the interactive Tutoring Calendar.
  • Design: To stand out, we chose a vibrant, energetic color palette and incorporated modern design elements like gradients and subtle animations. The layout is fully responsive, ensuring compatibility on desktops, tablets, and mobile devices, as per competition guidelines.
  • Implementation: The site is built as a single-page application using HTML, Tailwind CSS, and JavaScript. This approach provides a seamless user experience. Interactive elements like the tutoring calendar, quizzes, and learning games were implemented to maximize user engagement.

Measuring Success

To evaluate the effectiveness of this platform, we plan to track key metrics such as user engagement time, completion rates for quizzes and games, and the number of tutoring sessions booked through the calendar. This data will help us continuously improve the learning experience.
